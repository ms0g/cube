# cube
Wolfenstein 3D style raycasting engine

![raycaster](assets/raycaster.gif)
### Prerequisites
+ [CMake](http://www.cmake.org "CMake project page") (>= 3.23)
+ [SDL2](https://www.libsdl.org "Simple DirectMedia Layer") (>= 2.24.1)

### Building
```bash
cd build
./builder.sh
```

### Run
```bash
./cube
```
